## Verkefni 4 - Viðmótsforritun HBV201G
# Um verkefnið 
Þessi forrit geymir upplýsingar um útgjöld notanda. Öll útgjöld eru skipt niður með flokka, sem er hægt að bæta og eyða út (Þegar flokk er eydd, þá útgjöld í honum líka).  
Forrit sýnir summu alla flokka og hægt er að fara í flokk til að sjá útgjöld í því.  
Útgjöld er hægt að bæta (Nafn útgjalds, kostnaður og dagsetning) þegar farið er í flokk. Það er líka hægt að eyða þeim.

# Hvernig á að keyra verkefni?
Það er nóg að sækja forrit úr GitHub og keyra það með JavaFx run í Plugins (intellij).
